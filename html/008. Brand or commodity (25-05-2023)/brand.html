Yuhuuu, hyperparameter tunning nya udah siapp.

Hyperparameter tunning itu apa sih? Jadi ini tuh kayak menentukan parameter terbaik untuk suatu model machine learning yang kita bangun. Kayak misalkan model random forest, nanti ada parameter jumlah pohon (n_estimators=10) atau bisa 20, 30, 40, sama ada banyak parameter lain juga. Untuk setiap pemodelan kan yang dipake cuma satu angka per parameter, jadi hyperparameter tunning ini kita gunakan untuk memilih angka berapa sih yang akan menghasilkan output evaluasi score model paling tinggi.

Ohh iya, barusan aku juga ngubah test set data, dari sebelumnya perbandingan training : testing data 80:20, aku ubah jadi 60:40. Karena datanya dikit, cuma 300-an dan kakak mentornya nyaranin untuk ubah testnya jadi 40% agar hasil score evaluasi kami jadi lebih meyakinkan.
